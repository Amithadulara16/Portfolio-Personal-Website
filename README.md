# 🚀 DevByAmith Portfolio

A modern and responsive **Full-Stack Developer Portfolio Website** built using **Next.js, TypeScript, and Tailwind CSS**.
This project showcases my skills, projects, and contact information with a clean and premium UI design.

## 📦 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/devbyamith-portfolio.git
cd devbyamith-portfolio
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Run Development Server

```bash
npm run dev
```

👉 Open in browser:
http://localhost:3000

---

## 🏗️ Build for Production

```bash
npm run build
npm start
```

---

## 🧾 Customization

You can easily update:

* Profile image → `/public/images/`
* Projects → `ProjectCard.tsx`
* Skills → `SkillCard.tsx`
* CV → `/public/cv.pdf`
* Personal info → Navbar / Footer
